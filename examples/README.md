# Examples

Public-facing samples live here.

Current examples:

- `OkojoModuleSample` - small ES module graph and import/export behavior
- `OkojoModuleSampleRunner` - host runner for the module sample
- `OkojoGameLoopSandbox` - app/game-style embedding with host-owned frame budgets
- `OkojoHostEventLoopSandbox` - host scheduler and event-loop integration patterns
- `OkojoArtSandbox` - p5.js-like sketch sample on top of Okojo hosting APIs

Internal probes, debug apps, and bring-up experiments remain under `sandbox`.
